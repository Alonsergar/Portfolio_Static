Projectly

Este trabajo consiste en la creación de una página para aprender programar basada en proyectos y hecha con el stack MERN. A un usuario se le proporicona de la documentación necesaria y de unas pautas claras para cada proyecto en concreto. Luego el usuario puede compartir su proyecto en la plataforma. Hay que registrarse con google y github. Se crea una base de datos para almacenar a los diferentes usuarios y su avance en lo referente a los proyectos.

$ git clone https://github.com/Alonsergar/Projectly

Game of life

Una copia muy sencilla del game of life hecha en python.
Está desarrollado con pygame así requiere tenerlo instalado.
Al ejecutar el código nos va a aparecer el mapa de celdas y podremos poner que celdas están vivas y cuales no.
Luego al pulsar "s" iniciará la simulación.

$ git clone https://github.com/Alonsergar/Conways_GameOfLife.git
